00 — Introducción

Das Verb
WERDEN

werden es uno de los verbos más importantes del alemán. Significa «llegar a ser / convertirse», construye el futuro, la voz pasiva y el Konjunktiv II. Dominarlo es dominar la gramática alemana.

En esta guía: tres grandes usos, todos los tiempos verbales con conjugaciones completas, decenas de ejemplos y ejercicios interactivos con respuestas.

Verbo irregular Auxiliar de futuro Auxiliar de voz pasiva Konjunktiv II A2 → C1
01 — Usos principales

Los tres grandes usos de werden

🌱

1. Verbo léxico pleno

Expresa transformación o cambio de estado: «llegar a ser», «ponerse», «convertirse en».

Er wird Arzt. → Él se hace médico.
Es wird kalt. → Se pone frío.

🔮

2. Auxiliar de Futuro

Con infinitivo (Futur I) o infinitivo perfecto (Futur II) expresa futuro o probabilidad.

Sie wird kommen. → Vendrá.
Er wird geschlafen haben. → Habrá dormido.

🔄

3. Auxiliar de voz Pasiva

Con Partizip II forma el Vorgangspassiv. El Zustandspassiv usa sein.

Das Buch wird gelesen. → Es leído.
Das Fenster ist geöffnet. → Está abierta.


  • werden + Adj./Sust.
    Uso léxico: cambio de estado o identidad
  • werden + Infinitiv
    Futur I: acciones futuras o suposición sobre el presente
  • werden + Partizip II
    Vorgangspassiv: el sujeto recibe la acción

  • Ich werde müde.
    Me estoy poniendo cansado/a.
  • Du wirst Ingenieur, oder?
    Te vas a hacer ingeniero, ¿no?
  • Es wird dunkel.
    Se está oscureciendo.

03 — Präteritum

Pretérito Imperfecto Präteritum

La raíz es wurde-. Muy usado en la escritura y en la voz pasiva narrativa.

Singular
ichwurde
duwurdest
er/sie/eswurde
Plural
wirwurden
ihrwurdet
sie/Siewurden

  • Sie wurde Lehrerin.
    Ella se convirtió en profesora.
  • Es wurde kalt und windig.
    Se puso frío y ventoso (hizo viento).
  • Wir wurden nervös.
    Nos pusimos nerviosos.
  • Das Haus wurde gebaut.
    La casa fue construida. (voz pasiva)
  • Er wurde zum König gekrönt.
    Fue coronado rey.
  • Die Kinder wurden müde.
    Los niños se cansaron.

Se forma con sein + geworden. En la voz pasiva el ge- desaparece: solo worden.

wirsind … worden
ihrseid … worden
sie/Siesind … worden

  • Er ist Arzt geworden.
    Se ha convertido en médico.
  • Das Kind ist größer geworden.
    El niño ha crecido.
  • Das Gebäude ist renoviert worden.
    El edificio ha sido renovado. (voz pasiva)
  • Die Rechnung ist bezahlt worden.
    La factura ha sido pagada. (voz pasiva)

⚠️ geworden (léxico/futuro) vs worden (solo en voz pasiva Perfekt)

05 — Futur I

Futuro I werden + Infinitiv

Presente de werden + infinitivo al final de la oración. Tres usos principales:

📅

Futuro real

Acciones que ocurrirán. (El alemán también usa el Präsens + adverbio temporal.)

🤔

Probabilidad presente

Er wird schlafen → Seguramente está durmiendo ahora mismo.

🎯

Intención / promesa

Ich werde dich nie vergessen. → Nunca te olvidaré. (solemne)

  • Morgen werden wir nach Berlin fahren.
    Mañana iremos a Berlín.
  • Sie wird bestimmt gewinnen.
    Seguro que ganará.
  • Ich werde dir helfen.
    Te ayudaré. (promesa)
  • Du wirst es nicht glauben!
    ¡No te lo vas a creer!
  • Er wird jetzt zu Hause sein.
    Seguramente estará en casa ahora. (probabilidad)

📌 Sujeto + werden (conj.) + … + Infinitiv — Ich werde morgen früh aufstehen.

06 — Futur II

Futuro II werden + Infinitiv Perfekt

Acción que habrá ocurrido antes de un punto futuro, o suposición sobre algo ya sucedido.

werden (Präsens) + Partizip II + haben / sein
Estructura del Futuro II
Con haben
ichwerde … gemacht haben
duwirst … gemacht haben
er/sie/eswird … gemacht haben
wirwerden … gemacht haben
Con sein
ichwerde … gegangen sein
duwirst … gegangen sein
er/sie/eswird … gegangen sein
wirwerden … gegangen sein

  • Bis morgen werde ich den Bericht geschrieben haben.
    Para mañana habré escrito el informe.
  • Er wird schon angekommen sein.
    Seguramente ya habrá llegado. (suposición)
  • Sie werden es vergessen haben.
    Lo habrán olvidado.

  • Ich würde gern nach Japan reisen.
    Me gustaría viajar a Japón.
  • Wenn ich reich wäre, würde ich ein Haus kaufen.
    Si fuera rico, compraría una casa.
  • Das würde ich nie tun.
    Eso nunca lo haría.
  • Würdest du mir helfen?
    ¿Me ayudarías?
  • Wenn er geübt hätte, wäre er besser geworden.
    Si hubiera practicado, habría mejorado.
08 — Vorgangspassiv

Voz pasiva de proceso Vorgangspassiv

werden + Partizip II — describe la acción en sí misma. El agente se indica con von + Dativ.

ihrwerdet gefragt
sie/Siewerden gefragt
Präteritum Passiv
ichwurde gefragt
duwurdest gefragt
er/sie/eswurde gefragt
wirwurden gefragt
ihrwurdet gefragt
sie/Siewurden gefragt
Perfekt Passiv

dubist gefragt worden
er/sie/esist gefragt worden
wirsind gefragt worden
Futur I Passiv
ichwerde gefragt werden
duwirst gefragt werden
er/sie/eswird gefragt werden
wirwerden gefragt werden

  • Das Buch wird von dem Schüler gelesen.
    El libro es leído por el alumno. (Präsens)
  • Die Tür wurde geschlossen.
    La puerta fue cerrada. (Präteritum)
  • Der Brief ist geschrieben worden.
    La carta ha sido escrita. (Perfekt)
  • Das Paket wird morgen geliefert werden.
    El paquete será entregado mañana. (Futur I)
09 — Zustandspassiv

Voz pasiva de estado Zustandspassiv

sein + Partizip II — describe el estado resultante de una acción. La diferencia con el Vorgangspassiv es fundamental:

TipoFormaciónSignificadoEjemplo
Vorgangspassivwerden + Part.IIla acción en síDie Tür wird geöffnet. → alguien la abre
Zustandspassivsein + Part.IIel estado resultanteDie Tür ist geöffnet. → está abierta
  • Das Geschäft ist geschlossen.
    La tienda está cerrada. (estado)
  • Das Geschäft wird geschlossen.
    La tienda está siendo cerrada. (proceso)
  • Der Tisch ist gedeckt.
    La mesa está puesta.
  • Das Auto ist repariert.
    El coche está reparado.
  • Die Aufgabe ist erledigt.
    La tarea está completada.

11 — Übungen II

Ejercicios II Futuro y voz pasiva

🔮 D. Forma el Futuro I

  1. Morgen / ich / arbeiten → Morgen werde ich arbeiten.
  2. Sie / nächstes Jahr / heiraten → ___
  3. Wir / das Problem / lösen → ___
  4. Du / bestimmt / gewinnen → ___
  5. Er / vielleicht / krank sein → ___
  6. Es / bald / regnen → ___
2. Sie wird nächstes Jahr heiraten.
3. Wir werden das Problem lösen.
4. Du wirst bestimmt gewinnen.
5. Er wird vielleicht krank sein.
6. Es wird bald regnen.

🔄 E. Transforma al Vorgangspassiv

  1. Der Koch kocht das Essen. (Präsens) → ___
  2. Die Schüler lernen das Lied. (Präsens) → ___
  3. Die Polizei verhaftete den Dieb. (Präteritum) → ___
  4. Die Mutter hat den Brief geschrieben. (Perfekt) → ___
  5. Man wird das Museum renovieren. (Futur I) → ___
1. Das Essen wird (vom Koch) gekocht.
2. Das Lied wird (von den Schülern) gelernt.
3. Der Dieb wurde (von der Polizei) verhaftet.
4. Der Brief ist (von der Mutter) geschrieben worden.
5. Das Museum wird renoviert werden.

⚖️ F. wird o ist

  1. Das Licht _____ ausgeschaltet. (alguien lo apaga ahora)
  2. Das Licht _____ ausgeschaltet. (ya está en off)
  3. Der Kuchen _____ gebacken. (el proceso ocurre ahora)
  4. Der Kuchen _____ gebacken. (resultado: ya está listo)
  5. Das Auto _____ repariert. (el mecánico lo está arreglando)
1. wird (Vorgangspassiv)
2. ist (Zustandspassiv)
3. wird (Vorgangspassiv)
4. ist (Zustandspassiv)
5. wird (Vorgangspassiv)

🎯 I. Texto con huecos — todos los usos

Lena studiert Medizin und (1)_____ eines Tages Ärztin. Sie sagt: „Ich (2)_____ den Menschen helfen." Das Krankenhaus (3)_____ gerade renoviert. Bis nächstes Jahr (4)_____ es fertig renoviert worden _____. Wenn Lena mehr Zeit hätte, (5)_____ sie öfter Sport treiben.

1. wird — léxico (proceso de llegar a ser)
2. werde — Futur I (intención/promesa)
3. wird — Vorgangspassiv Präsens
4. wird … sein — Futur II Passiv (wird … worden sein)
5. würde — Konjunktiv II (würde … treiben)
